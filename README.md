# Weather Dashboard

## Project Overview

This Weather Dashboard is built using HTML, CSS and JavaScript.

It fetches live weather data from the OpenWeatherMap REST API using the Fetch API and async/await.

## Features

- Real-time weather
- Search by city
- Temperature
- Humidity
- Wind Speed
- Weather Condition
- Error Handling
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- REST API
- Async/Await
- JSON

## API Used

OpenWeatherMap API
